Le roman et le récit

Maîtrise les fondamentaux de l'analyse narrative pour l'épreuve de Français.

📄 Fiche de cours

Objet d'étude — Première Générale

1. Définition et caractéristiques du roman

Le roman est une œuvre narrative longue en prose qui raconte une histoire fictive ou inspirée du réel. Ses principales caractéristiques sont :

  • La longueur : contrairement à la nouvelle, il développe une intrigue complexe
  • La prose : il est écrit en prose, non en vers
  • Les personnages : ils sont construits psychologiquement et évoluent
  • La fiction : même inspiré du réel, il propose un monde imaginaire

2. Les types de narrateurs

Type
Pronoms
Caractéristiques
Narrateur interne (je)
Je / nous
Le narrateur est un personnage de l'histoire. Vision limitée.
Narrateur externe
Il / elle
Extérieur à l'histoire, il ne connaît que ce qu'il observe.
Narrateur omniscient
Il / elle
Sait tout des personnages, y compris leurs pensées.

3. Les points de vue narratifs

  • Point de vue interne : on voit l'histoire à travers les yeux d'un personnage
  • Point de vue externe : le narrateur décrit sans accéder aux pensées
  • Point de vue omniscient : le narrateur sait tout et voit tout
💡 Astuce bac : Pour identifier le point de vue, demande-toi : "Le narrateur a-t-il accès aux pensées des personnages ?"

4. L'incipit et l'explicit

L'incipit (début du roman) a plusieurs fonctions :

  • Présenter le cadre spatio-temporel
  • Introduire les personnages principaux
  • Créer un horizon d'attente pour le lecteur
  • Instaurer un pacte de lecture (réaliste, fantastique…)

L'explicit (fin du roman) peut être : ouvert, fermé, circulaire, ou in medias res.

5. Le temps dans le récit

  • L'analepse (retour en arrière / flashback)
  • La prolepse (anticipation / flashforward)
  • Le résumé : accélération du récit
  • La scène : temps de la narration = temps de l'histoire
  • La pause : description qui suspend l'action
  • L'ellipse : passage sous silence d'une période
