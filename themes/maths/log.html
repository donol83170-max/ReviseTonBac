Logarithme & exponentielle

Maîtrise les fonctions ln et exp, leurs propriétés, limites, équations et inéquations.

📄 Fiche de cours

Terminale Générale — Analyse

1. Fonction logarithme népérien ln

Domaine : ]0 ; +∞[  |  ln(e) = 1  |  ln(1) = 0

Propriété
Formule
Exemple
Produit
ln(ab) = ln(a) + ln(b)
ln(6) = ln(2) + ln(3)
Quotient
ln(a/b) = ln(a) − ln(b)
ln(1/2) = −ln(2)
Puissance
ln(aⁿ) = n × ln(a)
ln(8) = 3 ln(2)
Dérivée
(ln x)' = 1/x  ;  (ln u)' = u'/u
ln(x²+1) → 2x/(x²+1)

Limites :

lim(x→0⁺) ln(x) = −∞     lim(x→+∞) ln(x) = +∞
💡 La croissance de ln est très lente : ln "l'emporte" sur toute puissance de ln mais est "dominée" par tout xᵅ (α > 0).

2. Fonction exponentielle eˣ

Domaine : ℝ  |  Toujours positive  |  e⁰ = 1

Propriété
Formule
Exemple
Produit
eᵃ × eᵇ = eᵃ⁺ᵇ
e² × e³ = e⁵
Quotient
eᵃ / eᵇ = eᵃ⁻ᵇ
e⁵ / e² = e³
Puissance
(eᵃ)ⁿ = eⁿᵃ
(e²)³ = e⁶
Dérivée
(eˣ)' = eˣ  ;  (e^u)' = u' × e^u
e^(3x) → 3e^(3x)

Limites :

lim(x→−∞) eˣ = 0     lim(x→+∞) eˣ = +∞
💡 Croissances comparées : eˣ "l'emporte" sur tout polynôme : lim(x→+∞) eˣ/xⁿ = +∞. Et lim(x→−∞) xⁿeˣ = 0.

3. Fonctions réciproques

ln et exp sont des fonctions réciproques l'une de l'autre :

e^(ln x) = x   (pour x > 0)      ln(eˣ) = x   (pour tout x ∈ ℝ)

Résolution d'équations :

  • ln(x) = a ⟺ x = eᵃ   (appliquer exp des deux côtés)
  • eˣ = b ⟺ x = ln(b)   (appliquer ln des deux côtés, avec b > 0)

4. Étude de fonctions — rappels clés

  • ln est strictement croissante et concave sur ]0 ; +∞[ (f'' = −1/x² < 0)
  • eˣ est strictement croissante et convexe sur ℝ (f'' = eˣ > 0)
  • ln(x) < ln(y) ⟺ x < y  (inégalité conservée car ln croissante)
  • eˣ < eʸ ⟺ x < y  (inégalité conservée car exp croissante)
💡 Pour résoudre ln(f(x)) > ln(g(x)), il faut d'abord s'assurer que f(x) > 0 et g(x) > 0, puis comparer f(x) et g(x).
