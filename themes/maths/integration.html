💡 Règle : Deux primitives d'une même fonction diffèrent d'une constante. Si F est une primitive de f, toutes les primitives sont de la forme F(x) + C.

2. Intégrale définie

Si F est une primitive de f sur [a ; b], l'intégrale de a à b est :

∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) − F(a)

Propriétés :

  • Linéarité : ∫ₐᵇ (αf + βg) dx = α∫ₐᵇ f dx + β∫ₐᵇ g dx
  • Relation de Chasles : ∫ₐᵇ f dx = ∫ₐᶜ f dx + ∫ᶜᵇ f dx
  • Borne identique : ∫ₐᵃ f dx = 0
  • Inversion des bornes : ∫ᵇₐ f dx = −∫ₐᵇ f dx

3. Interprétation géométrique

Si f(x) ≥ 0 sur [a ; b], l'intégrale ∫ₐᵇ f(x) dx représente l'aire (en unités carrées) du domaine délimité par la courbe, l'axe des abscisses et les droites x = a et x = b.

Si f change de signe, l'aire géométrique s'obtient en séparant les zones positives et négatives :

Aire = ∫ₐᶜ f(x) dx − ∫ᶜᵇ f(x) dx   (si f ≥ 0 sur [a;c] et f ≤ 0 sur [c;b])
💡 Aire entre deux courbes : Si f ≥ g sur [a;b], l'aire entre les deux courbes est ∫ₐᵇ (f(x) − g(x)) dx.

4. Valeur moyenne

La valeur moyenne d'une fonction f sur [a ; b] est :

μ = 1/(b−a) × ∫ₐᵇ f(x) dx

Elle représente la "hauteur moyenne" de la courbe sur cet intervalle.

5. Fonctions paires et impaires

  • Si f est paire (f(−x) = f(x)) : ∫₋ₐᵃ f(x) dx = 2∫₀ᵃ f(x) dx
  • Si f est impaire (f(−x) = −f(x)) : ∫₋ₐᵃ f(x) dx = 0
💡 Reconnaître la parité d'une fonction permet de simplifier considérablement les calculs d'intégrales sur des intervalles symétriques.
