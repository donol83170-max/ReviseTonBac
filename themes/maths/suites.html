Suites Numériques

🔢 Arithmétique & Géométrique ⏱️ ~25 min de lecture 🎯 QCM inclus

1. Définitions fondamentales

Définition — Suite numérique Une suite numérique est une fonction définie sur ℕ (ou une partie de ℕ) à valeurs réelles. On note les termes u₀, u₁, u₂, ..., uₙ ou encore (uₙ)ₙ∈ℕ.

Deux modes de définition

  • Définition explicite : on exprime uₙ directement en fonction de n. Ex : uₙ = 2n + 1
  • Définition récurrente : on exprime uₙ₊₁ en fonction de uₙ (et donne un terme initial u₀ ou u₁).
Exemple — définition récurrente u₀ = 3 et uₙ₊₁ = uₙ + 5 pour tout n ∈ ℕ.
On calcule : u₁ = 8, u₂ = 13, u₃ = 18... C'est une suite arithmétique de raison 5.

2. Suites arithmétiques

Définition Une suite (uₙ) est arithmétique de raison r si pour tout n : uₙ₊₁ = uₙ + r

Formules clés

Terme général : uₙ = u₀ + n·r  |  uₙ = u₁ + (n-1)·r
Somme des n+1 premiers termes : Sₙ = (n+1) × (u₀ + uₙ) / 2

Sens de variation

  • Si r > 0 : suite croissante
  • Si r < 0 : suite décroissante
  • Si r = 0 : suite constante
Exemple complet u₁ = 7, raison r = 3.
• u₁₀ = u₁ + 9×3 = 7 + 27 = 34
• Somme S = u₁ + u₂ + ... + u₁₀ = 10 × (7+34)/2 = 10 × 20,5 = 205

3. Suites géométriques

Définition Une suite (uₙ) est géométrique de raison q (q ≠ 0) si pour tout n : uₙ₊₁ = q × uₙ

Formules clés

Terme général : uₙ = u₀ × qⁿ  |  uₙ = u₁ × qⁿ⁻¹
Somme (q ≠ 1) : Sₙ = u₀ × (1 - qⁿ⁺¹) / (1 - q)

Sens de variation (si u₀ > 0)

  • Si q > 1 : suite croissante
  • Si 0 < q < 1 : suite décroissante
  • Si q < 0 : suite alternativement positive et négative
Exemple complet u₀ = 2, raison q = 3.
• u₄ = 2 × 3⁴ = 2 × 81 = 162
• S₄ = 2 × (1 - 3⁵)/(1 - 3) = 2 × (-242)/(-2) = 2 × 121 = 242
Piège classique Pour la somme S = u₀ + u₁ + ... + uₙ, il y a n+1 termes (de l'indice 0 à n). Ne pas confondre n termes et n+1 termes !

4. Sens de variation

Comment étudier la monotonie ?

Pour montrer qu'une suite est croissante ou décroissante, on calcule uₙ₊₁ − uₙ :

  • Si uₙ₊₁ − uₙ > 0 : suite strictement croissante
  • Si uₙ₊₁ − uₙ < 0 : suite strictement décroissante
  • Si uₙ₊₁ − uₙ = 0 : suite constante

On peut aussi comparer uₙ₊₁ / uₙ à 1 (pour les suites à termes strictement positifs) :

  • Si uₙ₊₁ / uₙ > 1 : croissante
  • Si uₙ₊₁ / uₙ < 1 : décroissante

5. Limites et convergence

Définition — Convergence Une suite (uₙ) converge vers un réel L si uₙ tend vers L quand n tend vers +∞. On note : lim uₙ = L.

Limites usuelles à connaître

lim(n→+∞) qⁿ = 0 si |q| < 1  |  diverge si |q| ≥ 1 (q ≠ 1)
Toute suite arithmétique de raison r ≠ 0 diverge vers ±∞
Théorème des suites encadrées (gendarmes) Si aₙ ≤ uₙ ≤ bₙ et que lim(aₙ) = lim(bₙ) = L, alors lim(uₙ) = L.
