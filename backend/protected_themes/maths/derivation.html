2. Règles de dérivation

Somme :

(u + v)' = u' + v'

Produit :

(u × v)' = u'v + uv'

Quotient :

(u/v)' = (u'v − uv') / v²

Composée :

(f ∘ g)' = g' × (f' ∘ g)    ou    (u^n)' = n·u'·u^(n−1)
💡 Moyen mnémotechnique (quotient) : "La dérivée du haut fois le bas, moins le haut fois la dérivée du bas, le tout sur le bas au carré."

4. Tangente à une courbe

La tangente à la courbe de f au point d'abscisse x₀ a pour équation :

y = f'(x₀) · (x − x₀) + f(x₀)
  • f'(x₀) est le coefficient directeur (pente) de la tangente
  • f(x₀) est l'ordonnée du point de tangence

5. Dérivée seconde et convexité

La dérivée seconde f'' est la dérivée de f'.

  • Si f''(x) > 0 sur I → f est convexe sur I (courbe tournée vers le haut)
  • Si f''(x) < 0 sur I → f est concave sur I (courbe tournée vers le bas)
  • Si f'(x₀) = 0 et f''(x₀) > 0 → minimum local en x₀
  • Si f'(x₀) = 0 et f''(x₀) < 0 → maximum local en x₀
💡 Un point d'inflexion est un point où f'' s'annule et change de signe — la courbe change de concavité.
