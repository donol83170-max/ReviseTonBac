Terminale Générale — Probabilités

1. Loi binomiale B(n, p)

On répète n fois de façon indépendante une épreuve de Bernoulli de probabilité de succès p. Le nombre de succès X suit la loi binomiale :

P(X = k) = C(n,k) × pᵏ × (1−p)ⁿ⁻ᵏ

2. Loi normale N(μ, σ²)

La loi normale est une loi continue en cloche, symétrique autour de μ. La loi normale centrée réduite N(0,1) a μ = 0 et σ = 1.

Règle des 1-2-3 sigma :

  • P(μ − σ ≤ X ≤ μ + σ) ≈ 68 %
  • P(μ − 2σ ≤ X ≤ μ + 2σ) ≈ 95 %
  • P(μ − 3σ ≤ X ≤ μ + 3σ) ≈ 99,7 %

Réduction : si X ~ N(μ ; σ²), alors Z = (X − μ)/σ suit N(0,1).

💡 Approximation : B(n, p) ≈ N(np ; np(1−p)) quand np ≥ 5 et n(1−p) ≥ 5 (théorème de Moivre-Laplace).

3. Intervalle de fluctuation

Pour une proportion théorique p et un échantillon de taille n, l'intervalle de fluctuation au seuil 95% est :

I = [p − 1/√n ; p + 1/√n]

La fréquence observée f a environ 95% de chances de tomber dans cet intervalle.

  • Si f ∈ I → pas de raison de rejeter l'hypothèse
  • Si f ∉ I → on rejette l'hypothèse H₀ au seuil 5%

4. Intervalle de confiance

À partir d'une fréquence f observée sur un échantillon de taille n, on estime p par :

IC₉₅ = [f − 1/√n ; f + 1/√n]

On est "confiant à 95%" que la vraie proportion p est dans cet intervalle.

5. Probabilités — rappels essentiels

  • Formule d'inclusion-exclusion : P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
  • Indépendance : A et B indépendants ⟺ P(A ∩ B) = P(A) × P(B)
  • Probabilité conditionnelle : P(A|B) = P(A ∩ B) / P(B)
  • Espérance : E(X) = Σ xᵢ × pᵢ
  • Variance : V(X) = E(X²) − [E(X)]²  (formule de König-Huygens)
  • Écart-type : σ = √V(X)
