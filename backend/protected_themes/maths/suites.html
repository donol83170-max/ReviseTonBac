Suites numériques

Maîtrise les suites arithmétiques, géométriques et le raisonnement par récurrence.

📄 Fiche de cours

Terminale Générale — Analyse

1. Définition d'une suite

Une suite numérique est une fonction définie sur ℕ (ou une partie de ℕ) à valeurs dans ℝ. On note un le terme de rang n.

  • Suite explicite : un = f(n) — le terme est donné directement en fonction de n
  • Suite par récurrence : un+1 = f(un) — chaque terme est défini à partir du précédent

2. Suite arithmétique

Une suite (un) est arithmétique si chaque terme s'obtient en ajoutant une constante r (raison) au terme précédent.

un+1 = un + r

Terme général :

un = u0 + n·r    ou    un = up + (n−p)·r

Somme des n premiers termes :

S = n × (premier terme + dernier terme) / 2
💡 Moyen mnémotechnique : Suite arithmétique = addition constante. Pense à une suite de marches régulières.

3. Suite géométrique

Une suite (un) est géométrique si chaque terme s'obtient en multipliant par une constante q (raison) le terme précédent.

un+1 = un × q

Terme général :

un = u0 × qn    ou    un = up × q(n−p)

Somme des n premiers termes (q ≠ 1) :

S = u0 × (1 − qn) / (1 − q)

5. Raisonnement par récurrence

Pour démontrer qu'une propriété P(n) est vraie pour tout n ≥ n₀ :

  • Initialisation : Vérifier P(n₀)
  • Hérédité : Supposer P(n) vraie (hypothèse de récurrence) et démontrer P(n+1)
  • Conclusion : Par le principe de récurrence, P(n) est vraie pour tout n ≥ n₀
💡 Erreur classique : Ne pas oublier l'initialisation ! Sans elle, le raisonnement est invalide.
