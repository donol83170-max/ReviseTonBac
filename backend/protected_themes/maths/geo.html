Géométrie dans l'espace

Maîtrise les vecteurs 3D, les plans, les droites, les distances et les angles dans l'espace.

📄 Fiche de cours

Terminale Générale — Géométrie

1. Vecteurs dans l'espace

Un vecteur de l'espace a trois coordonnées (x ; y ; z).

Opération
Formule
Exemple
Vecteur AB
B − A = (x_B−x_A ; y_B−y_A ; z_B−z_A)
A(1;2;3), B(4;6;1) → (3;4;−2)
Norme ||u||
√(x² + y² + z²)
u(3;−4;0) → 5
Milieu M de [AB]
((x_A+x_B)/2 ; (y_A+y_B)/2 ; (z_A+z_B)/2)
A(0;0;0), B(2;4;6) → M(1;2;3)
Produit scalaire u·v
x₁x₂ + y₁y₂ + z₁z₂ = ||u||·||v||·cos(θ)
u(1;2;3)·v(4;−1;2) = 8
💡 Orthogonalité : u ⊥ v ⟺ u·v = 0. Colinéarité : u et v colinéaires ⟺ ∃k, u = k·v.

2. Plans dans l'espace

Un plan est défini par un point et un vecteur normal n(a ; b ; c). Son équation est :

a(x − x₀) + b(y − y₀) + c(z − z₀) = 0   soit   ax + by + cz = d
  • Le vecteur normal se lit directement dans les coefficients : n = (a ; b ; c)
  • Deux plans sont parallèles ⟺ leurs normaux sont colinéaires
  • Deux plans sont perpendiculaires ⟺ leurs normaux sont orthogonaux

Distance d'un point P(x₀ ; y₀ ; z₀) à un plan ax + by + cz + d = 0 :

dist = |ax₀ + by₀ + cz₀ + d| / √(a² + b² + c²)

3. Droites dans l'espace

Une droite est définie par un point A et un vecteur directeur u. Équations paramétriques :

x = x_A + t·u_x  ;  y = y_A + t·u_y  ;  z = z_A + t·u_z   (t ∈ ℝ)
  • Droite ⊥ à un plan ⟺ vecteur directeur colinéaire au vecteur normal du plan
  • Droite ∥ à un plan ⟺ vecteur directeur ⊥ au vecteur normal du plan
  • Droites gauches : ni parallèles, ni sécantes (pas dans le même plan)

4. Plan médiateur et sphère

Le plan médiateur de [AB] passe par le milieu de [AB] et est perpendiculaire à AB. Tout point M du plan médiateur vérifie MA = MB.

Une sphère de centre Ω(a ; b ; c) et de rayon r a pour équation :

(x−a)² + (y−b)² + (z−c)² = r²

5. Angles et orthogonalité

  • Angle entre deux vecteurs : cos(θ) = (u·v) / (||u||·||v||)
  • Angle entre deux plans : angle entre leurs vecteurs normaux
  • Angle entre une droite et un plan : complément de l'angle entre le vecteur directeur et le normal
💡 Deux droites de l'espace qui ne sont ni parallèles ni sécantes sont dites gauches : elles ne sont pas coplanaires.
